# /api/auth/logout

## DELETE - /api/auth/logout

Logs out a user.

### Parameters

| Name | Type | Description |
|------|------|-------------|
| none | none | none        |

### Body

| Name | Type | Description |
|------|------|-------------|
| none | none | none        |

### Response Codes

| Code | Description            |
|------|------------------------|
| 200  | Logs out a user.       |
| 500  | Internal Server Error. |

### Response

Deletes the session token.
